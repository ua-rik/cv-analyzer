# CV Analyzer

Next.js App Router застосунок, який приймає резюме, оцінює їх за індивідуальними критеріями та повертає бали від LLM.

## Локальний запуск

```bash
npm install
npm run dev
```

Далі відкрий `http://localhost:3000`.

## Змінні оточення

Створи `.env.local` з ключами (приклад):

```
OPENAI_API_KEY=sk-your-key
LLM_MODEL=gpt-4.1-mini
LLM_MAX_CONCURRENCY=3
```

## Як це працює

1. На клієнті вводяться до 10 критеріїв (назва, опис, вага) та API-ключ.
2. Файли відправляються на `/api/upload`, де зберігаються тимчасово у `/tmp/<sessionId>/`.
3. `/api/evaluate` парсить файли (`pdf-parse`, `mammoth`, plain text), відправляє текст одним запитом до LLM для кожного резюме, обмежуючи паралельність.
4. Повертається таблиця з балами, зваженим результатом та нотатками.

Ключ не зберігається, передається тільки в межах активної сесії запиту.
